> A modern and responsive **Library Management System** built for colleges and institutions to manage books, students, employees, and library operations efficiently.

---

## ✨ Features

### 👨‍🎓 Student Portal
- Browse available books
- Search by title, author, or category
- Issue & return requests
- Track issued books
- View due dates and history

### 👨‍💼 Employee Portal
- Manage book circulation
- Handle issue/return operations
- Monitor student requests
- Generate reports

### 👨‍💻 Admin Portal
- Full system control
- Add / Edit / Delete books
- Manage users and employees
- Track system analytics
- Monitor inventory

### 📊 Dashboard
- Real-time library statistics
- Total books
- Issued books
- Active users
- System health monitoring
